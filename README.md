# Thrive Market — Executive AI Training Exercises

Personalized hands-on exercise packages for Thrive Market's executive off-site Day 2 workshop. Each of the 6 C-suite executives gets a folder with 3 paste-ready Claude Code exercises, synthetic sample data, and a facilitator README.

## Structure

```
{INITIALS}/
├── README.md                    # Facilitator guide: profile, complexity, support notes
├── 01-{exercise-slug}/
│   ├── prompt.md                # Paste-ready prompt for Claude Code
│   └── *.csv / *.json / *.md   # 2-4 synthetic data files
├── 02-{exercise-slug}/
│   └── ...
└── 03-{exercise-slug}/
    └── ...
```

## How to Use

1. Navigate to the executive's folder
2. Read the `README.md` for facilitator context
3. Open the exercise folder (start with `01-`)
4. Upload the sample data files to Claude Code
5. Paste the contents of `prompt.md`
6. Let the exec iterate using the Iteration Hooks and Stretch Goals in the prompt
