# Scott Lescher (COO) — Exercise Packages

## Facilitator Guide

Scott is COO at Thrive Market, an e-commerce grocery company (organic/natural products) with 3 fulfillment centers (Batesville IN, Reno NV, and a third). 100 non-eng staff already use Claude. Execs are new to Claude Code.

### Exercises at a Glance

| # | Exercise | Time | What Scott Builds |
|---|----------|------|-------------------|
| 01 | Demand Exception Reviewer | 30-45 min | Exception report flagging forecast outliers with root cause + recommended action |
| 02 | Carrier Scorecard Builder | 30-45 min | One-page scorecards per carrier + network comparison + QBR talking points |
| 03 | FC Shift Handoff & Operations Brief | 30-45 min | Structured shift handoff with performance summary, prioritized tasks, staffing recs |

### How to Run

1. Each exercise folder contains a `prompt.md` and 2-4 synthetic data files
2. Scott drags the data files into Claude Code (or points to the folder)
3. He pastes the prompt from `prompt.md`
4. Claude Code builds the deliverable; Scott iterates using the Iteration Hooks at the bottom of each prompt

### Facilitation Tips

- **Start with Exercise 1** — it's the most structured and produces a clear table output. Good confidence builder.
- **Exercise 2** is the most visually rich — encourage Scott to ask for HTML output or charts.
- **Exercise 3** is the most operationally urgent — it maps to something FC managers do every single shift. Good closer.
- All data cross-references within each exercise (e.g., SKUs in forecast match SKUs in accuracy history). Encourage Scott to ask Claude to validate consistency.
- If time is short, skip stretch goals. They exist for participants who finish fast.

### Data Notes

- All data is synthetic but modeled on realistic e-commerce fulfillment patterns
- Carrier names are real (FedEx, UPS, etc.) but all performance data is fabricated
- SKU names reference organic/natural products consistent with Thrive Market's catalog
- FC locations reference Batesville IN and Reno NV (real Thrive facilities)
